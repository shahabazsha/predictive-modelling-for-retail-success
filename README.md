# 📊 Predictive Modelling for Retail Success

⭐ Final Year Project – Predictive Modelling for Retail Success  

---

## 📌 Overview
This project focuses on predicting retail sales using machine learning techniques. The system analyzes customer and transaction data to estimate total sales, helping businesses make better decisions.

---

## 🎯 Objectives
- Predict total sales using input features  
- Understand data preprocessing and model training  
- Demonstrate machine learning workflow  

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- Scikit-learn  
- Jupyter Notebook  

---

## 📂 Project Structure
- dataset.csv → Input data  
- project.ipynb → Jupyter Notebook (main code)  
- report.pdf → Final project report  

---

## ⚙️ How It Works
1. Load dataset using pandas  
2. Clean and preprocess data  
3. Select features (Age, Quantity, Price)  
4. Split dataset into training and testing  
5. Train model using Linear Regression  
6. Generate predictions  
7. Display output in table format  

---

## 🔄 Workflow
Input Data → Data Processing → Model Training → Prediction → Output  

---

## 📈 Model Used
- Linear Regression
